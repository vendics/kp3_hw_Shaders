# Декомпозиция и повторение анимации

Этот репозиторий содержит решение задачи по декомпозиции и повторению анимации из видеофайла.

## Условия задачи
Задача А. Декомпозировать и повторить анимацию

Дано:
- Видеофайл, содержащий анимацию.

Требуется:
1. Выполнить декомпозицию анимации, т.е. описать:
   - Элементы анимации.
   - Взаимосвязи между элементами.
   - Реализацию этих элементов.
2. Написать программу для повторения анимации согласно выполненной декомпозиции.

## Структура репозитория
- Shaders.ipynb — код программы для повторения анимации.
- data/ — папка с исходными данными, видеофайлом.
